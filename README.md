Тестовое задание для SaaSoft на позицию Middle Frontend программист-разработчик (Vue.js)

Ссылка на тестовое задание - https://docs.google.com/document/d/1sOLmhOWItVELDHl4AQ75PrzfByj5uK02uH2QMjQ0xuM/edit?usp=sharing

Стек текущего проекта:
Vue.js (Composition API), TypeScript, Pinia, NaiveUI как UI-kit.

Для сохранения состояния между перезагрузками страницы используется синхронизация состояния Pinia с sessionStorage.
При инициализации стора данные восстанавливаются из хранилища браузера, при изменениях — автоматически сохраняются.
